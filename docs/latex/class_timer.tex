\doxysubsection{Opis szczegółowy}
Klasa \doxylink{class_timer}{Timer} umożliwia wyświetlanie aktualnego czasu na etykiecie. 

Ta klasa korzysta z obiektu QTimer do okresowego aktualizowania wyświetlanego czasu na etykiecie QLabel. Czas jest aktualizowany w sposób automatyczny co sekundę.
