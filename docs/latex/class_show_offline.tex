\doxysubsection{Opis szczegółowy}
Klasa odpowiedzialna za wyświetlanie danych pomiarowych zapisanych lokalnie (offline). 

Klasa odczytuje pliki JSON z katalogu "{}data/"{}, filtruje dane, umożliwia przeglądanie listy stacji i sensorów oraz generuje wykresy z podstawowymi statystykami dla wybranego sensora.
