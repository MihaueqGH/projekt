Opis szczegółowy

Klasa ApiManager odpowiada za pobieranie danych z API.

Klasa udostępnia metody statyczne do pobierania danych JSON z podanego URL-a, z obsługą błędów za pomocą callbacków.

Wysyła żądanie HTTP pod podany URL i zwraca dane w postaci QJsonArray.

Parametry

callbackFunkcja wywoływana po pomyślnym pobraniu danych.
errorCallbackFunkcja wywoływana w przypadku błędu (np. brak połączenia, zły URL).
